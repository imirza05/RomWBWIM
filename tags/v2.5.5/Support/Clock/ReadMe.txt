This directory contains the source and assembled versions of the ZSystem Clock Drivers for N8VEM & N8.

The .MAC files can be compiled using M80 with a command line such as "M80 =N8VEMCLK.MAC".

LDTIM_N8VEM.COM and LDTIM_N8.COM are copies of LDT for N8VEM and N8 respectively.  They were constructed useing SETUPZST using the clock drivers included here.  They are both setup to load as Resident System Extentions with support for DateStamper style date stamping.
